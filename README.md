# CSV_Manager_and_Plotter<br />
<Python, GUI><br />
Manage and plot .CSV files<br />
Run: CSV_Manager_and_Plotter.py<br />
CSV Manager: View, extract, convert delimiter, plot

Usage:<br />
>Peak: Displays the first rows of the .csv file to view its content and structure.<br />
>Rows to Peak: Choose the numnber of rows to "Peak".<br />
>Columns to Process: Choose all or specific columns to process.<br />
>Timestamp at Column 0: Choose this option to extract timestamp from column 0 in the .csv file.<br />
>Include Timestamp: Add timestamp to the output .csv file.<br />
>Action: Process Only: Extract columns and rows and prepare to save the new .csv file without plotting.<br />
>Action: Process and Plot: Extracts data and plots the chosen columns.<br />
